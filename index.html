VI Simple Slider

설명
터치/마우스 스와이핑 기능, 끝에서 처음으로 계속 반복으로 이어짐, 오토플레이 가능
사용법 예제

<div id="testDiv">
      <ul>
             <li data-link="클릭시 이동할 주소" data-target="링크 이동 타겟">
                  <img src="이미지 경로">
            </li>
            <li data-link="/test.html" data-target="_blank">
                  <img src="./img/slide05.jpg">
            </li>
      </ul>
      <a href="#" class="arrowBtn prev">이전</a>
      <a href="#" class="arrowBtn next">다음</a>
</div>

<script>
$('#testDiv').viSimpleSlider({
      ease : 'easeInOutQuart',
      autoPlay : true,
      autoTime : 3000,
      speed : 400
});
<script>

<script>
//다중 슬라이드로 구현할 경우 인터벌 충돌이 될 수 있으므로 클리어 인터벌 액션을 아래처럼 활용해주세요.
$('#testDiv').viClear();
<script>

옵션
ease
값 : String (기본값 : 'swing')
슬라이드 모셔닝
indicate
값 : true, false (기본값 : true)
슬라이드 목록 노출 여부
arrow
값 : true, false (기본값 : true)
좌우 화살표 노출 여부, true로 설정한 상태에서 만들어놓은 버튼이 없으면 자동 생성,
직접 버튼을 만들어 사용할 경우 기능을 부여하고 싶다면, 이전 버튼은 class="arrowBtn prev" 추가, 다음 버튼은 class="arrowBtn next" 추가
autoPlay
값 : true, false (기본값 : false)
자동 슬라이드
autoTime
값 : Number (기본값 : 3000)
얼마 간격으로 슬라이드 되는 지의 시간값, 1초 = 1000
speed
값 : Number (기본값 : 800)
슬라이드 넘어가는 속도
loop
값 : true, false (기본값 : true)
슬라이드를 끝에서 다시 처음으로 이어지게 하기
mobileSwipe
값 : true, false (기본값 : true)
모바일의 경우 터치 드래그로 스와이핑을 사용할 것인지 여부(터치 기능이 있어야 유효함)
desktopSwipe
값 : true, false (기본값 : true)
데스크탑의 경우 드래그해서 스와이핑을 사용할 것인지 여부
